# Code of Conduct

This project adopts version 2.1 of the [Contributor Covenant] as its
code of conduct. By participating in any space related to this project
(issues, pull requests, discussions, security reports), contributors and
maintainers agree to uphold its standards.

The full Covenant text — Pledge, Standards, Enforcement Responsibilities,
Scope, Enforcement, and Enforcement Guidelines — is canonical and lives
at the link below. Adopting by reference (rather than copying the text)
keeps this project aligned as the Covenant evolves.

[Contributor Covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

To report a concern under this Code of Conduct, please contact the
maintainer privately. Two channels are available:

- Open a [GitHub private security advisory] on this repository and
  prefix the title with `[conduct]`. This is the most reliable channel
  and works even if you don't already share contact details with the
  maintainer.
- Reach out via the maintainer's [GitHub profile] for a direct message.

[GitHub private security advisory]: https://github.com/entropyvortex/ai-consensus-mcp/security/advisories/new
[GitHub profile]: https://github.com/marceloceccon

Reports are confidential. Acknowledgement and triage follow the same
service levels as security reports — see [`SECURITY.md`](./SECURITY.md):
acknowledgement within 5 business days, triage decision within 10.
